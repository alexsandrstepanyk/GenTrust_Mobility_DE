## 🔧 Технічні деталі

### Використані технології:
- **React Native** - мобільний фреймворк
- **Expo SDK 54** - розробка та deployment
- **react-native-maps** - відображення карти з маркерами
- **axios** - HTTP запити до API
- **expo-secure-store** - збереження токена авторизації
- **i18next** - локалізація (3 мови)
- **React Navigation** - навігація між екранами
- **TypeScript** - типізація

### API Endpoints:
```
GET  /api/quests/available     - Список доступних квестів
GET  /api/users/active-quest   - Активне завдання користувача
POST /api/quests/{id}/take     - Прийняти завдання
```

### Конфігурація:
```typescript
// mobile-school/config.ts
API_URL = http://192.168.178.34:3000/api

// Доступ з телефону через WiFi
// Expo Tunnel mode для синхронізації
```

## 🚀 Як тестувати

### 1. Запустити сервери:
```bash

npm run api

# Термінал 2: Mobile App
cd mobile-school
npm start
```

### 2. На телефоні:
```
1. Відкрити Expo Go
2. Сканувати QR код
3. Логін: admin@test.com / admin
4. Перейти на вкладку Tasks
5. Натиснути "Get order" на будь-якому квесті
```
