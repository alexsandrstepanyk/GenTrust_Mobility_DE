## 🔧 ЯКЩО ЩОС НЕ ПРАЦЮЄ

### Проблема: "Cannot connect to API" на телефоні

**Перевірте:**
1. Комп'ютер і телефон на ОДНІЙ WiFi мережі
2. IP адреса правильна: `192.168.178.34`
3. Backend запущений:
   ```bash
   lsof -i :3000  # Має показати процес на порту 3000
   ```

### Проблема: Expo не знаходить додаток

1. Перевірте що додаток запущена:
   ```bash
   tail -f /tmp/expo.log
   ```
2. Спробуйте знову: Shake телефон → Reload

### Проблема: "Port already in use"

```bash
# Вбийте всі процеси
killall -9 npm node

# Перезапустіть
bash start_dev.sh
```
