## 🐛 ЯКЩО МОВА НЕ ЗБЕРІГАЄТЬСЯ

### Перевірте логи:

```bash
tail -f /tmp/expo-parent.log
```

### Шукайте:

```
[i18n] Language saved: uk
[i18n] Loaded saved language: uk
[App] i18n initialized, loaded language: uk
```

### Якщо немає логів:

1. Перевірте що `services/i18n.ts` має `export const saveLanguage`
2. Перевірте що `App.tsx` має `loadSavedLanguage()`
3. Перевірте що `ProfileScreen.tsx` використовує `saveLanguage()`
