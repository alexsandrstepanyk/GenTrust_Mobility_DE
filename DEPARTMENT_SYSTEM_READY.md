## 🚀 ШВИДКИЙ СТАРТ

### **1. Перезапустіть Backend**

```bash
# Вбити старий процес
lsof -ti:3000 | xargs kill -9

# Запустити новий
cd /Users/apple/Desktop/GenTrust_Mobility_DE
npx nodemon --exec ts-node src/api-server.ts
```

### **2. Перезапустіть City-Hall Dashboard**

```bash
cd city-hall-dashboard
npm install  # якщо потрібно
npm run dev
```

### **3. Відкрийте City-Hall Dashboard**

```
http://localhost:5173
```

### **4. Перейдіть в "Департаменти"**

- Кнопка в меню зліва
- Або безпосередньо `/departments`

### **5. Натисніть "Створити департамент"**

- Кнопка в правому верхньому куті
