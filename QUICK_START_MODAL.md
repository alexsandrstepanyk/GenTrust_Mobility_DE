## ✅ Що готово

Реалізовано професійний потік прийняття завдань як у Glovo:
- ⚠️ Попередження "Це все дуже серйозно"
- 🗺️ Карта з локацією завдання
- 📝 Детальний опис та умови
- 💰 Інформація про винагороду
- ✅ Двохкрокове підтвердження

---

## 🚀 ЯК ТЕСТУВАТИ (3 кроки)

### 1️⃣ Перевір що сервери запущені

```bash
# Перевірка API (порт 3000)
curl http://localhost:3000/health
# Має показати: {"status":"ok",...}

# Якщо не працює:
npm run api
```

### 2️⃣ Відкрий Expo на телефоні

```
Expo вже запущений на порту 8081 (tunnel mode)

На телефоні:
1. Відкрий Expo Go
2. Скануй QR код з терміналу
3. Дочекайся завантаження
```

### 3️⃣ Тестуй модаль

```
1. Логін: admin@test.com / admin
2. Вкладка "Tasks" (📦)
3. Натисни "Get order" на будь-якому квесті
4. Побачиш попередження ⚠️
5. "Прочитати детально" → деталі + карта
6. "ПРИЙНЯТИ ЗАВДАННЯ" (натисни 2 рази)
7. Перевір що квест перемістився в "Active Quest"
```

## 🐛 Якщо щось не працює

### API не відповідає
```bash
# Перезапусти API
killall node
npm run api
```

### Expo не підключається
```bash
# Перезапусти Expo
cd mobile-school
killall expo
npm start
```

### Модаль не відкривається
```
1. Перевір що увійшов (логін admin)
2. Перевір що є доступні квести
3. Перезавантаж додаток (shake phone → Reload)
```

### Карта не показується
```
На телефоні:
1. Дай дозвіл на GPS
2. Перезавантаж додаток
```
